A comprehensive hostel management application for IIT Hyderabad.

## Features

- Room allocation and management
- Maintenance request tracking
- Announcements and notifications
- Resident information system

## Tech Stack

- Svelte for lightweight and fast UI
- Tailwind CSS for beautiful styling

This project has been completed and is now managing multiple hostel operations.
